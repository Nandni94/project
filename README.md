💬 Chatting Website (MERN)

A real-time chat application where users can securely communicate with each other.

🚀 Features
User Authentication (Login/Signup)
Real-time Messaging (Socket.io)
Online/Offline Status
Responsive Design
🛠️ Tech Stack
Frontend: React.js, CSS
Backend: Node.js, Express.js
Database: MongoDB
Other: JWT, Socket.io


Install dependencies:

cd server && npm install
cd ../client && npm install

Run app:

cd server && npm start
cd ../client && npm start
